# Rishvin Reddy - Personal Portfolio

Welcome to the official portfolio repository of **Erolla Rishvin Reddy**, a B.Tech CSE student at Woxsen University specializing in **Blockchain, IoT, and Cybersecurity**. 

🌍 **Live Website:** [rishvin-reddy-portfolio.vercel.app](https://rishvin-reddy-portfolio.vercel.app/)

## 🚀 About the Project

This is a modern, responsive, and interactive personal portfolio built to showcase my projects, skills, and experience across various domains including software engineering, security, IoT, blockchain, full-stack development, and automation.

### 🛠 Tech Stack
- **Framework:** [Next.js](https://nextjs.org/) (App Router)
- **Library:** [React](https://react.dev/)
- **Styling:** [Tailwind CSS v4](https://tailwindcss.com/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/)
- **Icons:** [Lucide React](https://lucide.dev/)
- **Deployment:** [Vercel](https://vercel.com/)

## 🌟 Key Features
- **Modern UI/UX:** Clean and premium design with smooth micro-animations.
- **Responsive Design:** Fully optimized for mobile, tablet, and desktop screens.
- **SEO Optimized:** Structured metadata and Google Search Console verification included.
- **Dynamic Routing:** Built utilizing the Next.js App Router.

## 💻 Getting Started (Local Development)

To run this project locally, follow these steps:

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/RishvinReddy/rishvin-reddy-portfolio.git
   ```
2. Navigate to the project directory:
   ```bash
   cd rishvin-reddy-portfolio
   ```
3. Install dependencies:
   ```bash
   npm install
   # or
   yarn install
   # or
   pnpm install
   ```
4. Run the development server:
   ```bash
   npm run dev
   # or
   yarn dev
   # or
   pnpm dev
   ```
5. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
